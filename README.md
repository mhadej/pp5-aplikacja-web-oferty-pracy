# JobHunt
[PL version below]

JobHunt is a web application for browsing, filtering, and comparing job offers, created as a coursework project for the **Pracownia Programowania V** course.

## Features
- Two job categories: general jobs and IT jobs.
- Filtering by location, contract type, salary range, and requirements.
- Saving offers as favorites using localStorage.
- Comparing up to 3 offers at once in a clear comparison table.
- Light and dark mode with a toggle in the header.

## Technologies
- Plain **HTML, CSS, JavaScript (ES6+)**.
- Web Components (Custom Elements + Shadow DOM).
- Mocked job offers API provided by Mockaroo.

JobHunt jest aplikacją webową do przeglądania, filtrowania i porównywania ofert pracy, przygotowana jako projekt zaliczeniowy na przedmiot **Pracownia Programowania V**.

## Funkcje
- Dwie kategorie ofert: prace ogólne i IT.
- Filtrowanie po lokalizacji, typie umowy, widełkach płacowych oraz wymaganiach.
- Zapisywanie ofert jako ulubione z wykorzystaniem localStorage.
- Porównywanie maksymalnie 3 ofert jednocześnie w czytelnej tabeli.
- Tryb jasny i ciemny z przełącznikiem w nagłówku.

## Technologie
- Czyste **HTML, CSS, JavaScript (ES6+)**.
- Web Components (Custom Elements + Shadow DOM).
- Mockowane API Mockaroo z danymi ofert pracy.
